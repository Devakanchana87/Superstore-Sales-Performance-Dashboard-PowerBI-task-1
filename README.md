#  Superstore Sales Performance Dashboard — Power BI

## Overview
Superstore sales analysis dashboard built using Power BI
covering revenue trends, top products, and regional performance.

## Tools Used
- Power BI Desktop
- Microsoft Excel

## Superstore Dataset
- 300 sales orders | Jan–Dec 2024
- 4 Categories | 5 Regions | 20+ Products

## Key Insights
1. Electronics leads revenue at 47% (₹6M+)
2. East region dominates with 38.28% share (₹4.84M)
3. Rahul Verma is top salesperson
4. Monitor 27inch is best-selling product
5. Revenue peaked February 2024, steady growth from July
6. Overall Profit Margin is 92.7%
